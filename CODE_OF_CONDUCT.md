# Code of Conduct

This project uses a simple standard: be direct, technical, and respectful.

Do:

- Discuss ideas, evidence, tests, and tradeoffs.
- Give actionable feedback.
- Assume contributors are trying to improve the project unless behavior proves otherwise.

Do not:

- Harass, threaten, or target people.
- Submit deceptive work, fake evidence, copied proprietary code, malware, or credentials.
- Pressure maintainers to merge unverified changes.

Maintainers may remove comments, close issues, reject pull requests, or block contributors who violate this standard.
